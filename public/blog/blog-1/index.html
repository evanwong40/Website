Hello world. Here are some cool things that I learned in python in SDS 348!

The first neat thing that we talked abot was using Regular expressions functions in python to mine phrases and lines! In homework 11, we were asked to create certain outputs by using regex functions on a provided string. My favorite part was using these regex functions to find the restriction sites for a specific enzyme because this was my first real exposure to how we can use coding to find something useful in relation to biology. Up until this point in the course, I felt that alot of the things that we were doing in R were very statistics based, and had no clear relation to biological analysis. While learning about data manipulation through dplyr, tidyr, and ggplot functions was helpful and neat, there was really no real analysis being done. Unit 2 covered mainly linear regressin and modeling, which was a good segway into analytical techniques, but I never felt a strong connection to the “biology” aspect of the course besides maybe modeling health factors.

However, this problem above is a clear example of how biologists can use these coding techniques in their own research. I can almost picture myself extracting DNA from a living specimen, extracting the DNA through lysing of the membrane and centrigufation, inserting the dna into a genome sequence reader, and coding for a speicifc sequence in the DNA for restriction enzyme sites!
